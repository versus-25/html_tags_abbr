Тег Расшифровка Перевод
<a> anchor Якорь. Устанавливает якорь или ссылку на другую страницу.
<abbr> abbreviation Сокращение. Указывает, что текст является аббревиатурой.
<address> address Адрес. Определяет контактную информацию владельца сайта, статьи или документа.
<acronym> acronym Акроним. Не поддерживает в HTML5, аналог — <abbr>.
<area> area Область. Определяет область внутри карты изображений — <map>.
<article> article Статья. Определяет независимое, самостоятельное содержание, которое может быть вырвано из текста без потери смысла.
<aside> aside В стороне.  Используется для выделения содержимого связанного со страницей, но которое может рассматриваться отдельно.
<audio> audio Аудио. Определяет звуковой файл.
<b> bold Жирный. Определяет соответствующее начертание.
<base> base База. Указывает базовый адрес страницы URL.
<bdi> Bi-Directional Isolation Двунаправленная изоляция. Запрещает изменять направление текста.
<bdo> Bi-Directional Override Двунаправленный текст. Управляет направлением текста.
<big> big Большой. Использовался для увеличения шрифта. Не поддерживается HTML5, используйте CSS.
<blockquote> blockquote Блок цитаты. Определяет раздел, в котором приводится цитата из другого источника.
<body> body Тело. «Тело» документа. Фактически это весь контент документа.
<br> break Разрыв. Устанавливает разрыв строки.
<button> button Кнопка. Определяет кнопку элементом.
<canvas> canvas Холст. Используется для создания графики с помощью скриптов.
<caption> caption Подпись. Указывает заголовок таблицы.
<center> center Центр. Использовался для выравнивания текста по центру. Не используется в HTML5, для этого используйте CSS.
<cite> cite Ссылаться. Ссылается на произведение: книгу, песню и др.
<code> code Код. Используется для определения программного кода.
<col> columns Столбцы. Задает характеристики для одного или нескольких столбцов.
<colgroup> colgroup Группа столбцов. Определяет группу из одного или нескольких столбцов для форматирования.
<datalist> data list Список данных. Задает список предопределенных опций.
<dd> Description list description Описание списка описаний. Используется для описания термина в списке описаний.
<del> delete Удаленный. Определяет текст, удаленный из документа.
<details> details Подробности. Указывает дополнительные сведения о том, что пользователь может скрыть или посмотреть.
<dfn> definition Определение. Указывает экземпляр термина.
<dialog> dialog Диалог. Создает диалоговое окно.
<dir> dir Каталог. Использовался для отображения названия каталогов. Не используется в HTML5, для этого используйте CSS.
<div> division Разделение. Определяет раздел в документе. Я привык называть его блок.
<dl> description list Список сгруппированных элементов.  Содержит термины и описания.
<dt> description list term Группируется с описанием (элемент <dd>) в описывающих списках.
<em> emphasis Акцент. Подчеркивание слова или фразы в предложении
<embed> plugin point Подключение. Позволяет вставить внешнюю ссылку или приложение (не HTML) на страницу.
<fieldset> form field set Набор полей формы. Используется, чтобы сгруппировать связанные друг с другом формы управления и метки.
<figcaption> figure caption Подпись. Сопровождает элемент <figure>.
<figure> figure Фигура. Используется для группирования элементов иллюстрации (изображение, цвет, подпись).
<font> font Шрифт. Представляет собой контейнер для изменения различных характеристик шрифта.
<footer> footer Нижний колонтитул. Создает своеобразный «подвал» страницы где находится контактная информация, имя владельца и др.
<form> form Форма. Создает форму..
<frame> frame Фрейм, рамка. По сути — позволяет создать  веб-страницу в веб-странице.
<frameset> frame set Задать рамку. Определяет структуру фреймов на веб-странице.
<h1> headling 1-6 Заголовок. H1 – заголовок первого уровня, наиболее важный.
<head> head Голова. Тег в котором размещаются метаданные, подключаются стили и скрипты.
<header> header Заголовок. Тег, задающий «шапку» сайта или страницы. Как правило содержит область навигации.
<hgroup> head group Группа заголовков. Тег, обозначающий группу заголовков веб-страницы или раздела.
<hr> horizontal rule Горизонтальная линия. Рисует горизонтальную линию, вид которой зависит от заданных параметров.
<html> html Корневой элемент. Контейнер, который содержит все элементы веб-страницы.
<i> italic Курсив. Устанавливает курсивное написание фразы или предложения.
<iframe> inline frame Альтернативный фрейм. Создает плавающий фрейм, который находится внутри документа.
<img> image Изображение. Предназначен для отображения на странице файлов в графическом формате .JPG, .PNG и др.

Список атрибутов тега img.

<input> input Ввод. Данный тег позволяет создавать элементы интерфейса (текстовые поля, переключатели, кнопки, флажки).
<ins> insertion Внедрение. Позволяет выделять текст, чтобы проследить какие изменения были внесены в новую версию.
<isindex> is index Индекс. Предназначен для поискового индекса в текущем документе. Размещается внутри контейнера <head>.
<kbd> keyboard input Ввод с клавиатуры. Используется для обозначения текста, который набирается на клавиатуре.
<keygen> key generation Генератор ключей. Тег, генерирующий пару ключей — закрытый и открытый.
<label> label Метка. Устанавливает связь между меткой (к примеру текст) и элементом формы.
<legend> legend Надпись. Создает заголовки для группы элементов формы.
<li> list item Элемент списка. Определяет отдельный элемент списка.
<link> link Ссылка. Позволяет установить связь с внешними элементами вроде файлов со шрифтами или стилями и при этом не создает ссылку.
<listing> listing Список, а точнее код. Предназначен для вывода списка программ или кода.
<map>  map Карта. Задает карту изображений.
<mark> marked Отмеченный. Отмечает текст как выделенный. Можно изменить его вид с помощью стилей.
<marquee> marquee Шатер. Позволяет создать бегущую строку на странице.
<menu> menu Меню. Предназначен для отображения списка пунктов меню.
<meta> meta data Мета данные. Определяет метатеги, которые хранят информацию, предназначенную для браузеров и поисковых систем.
<meter> meter Метр. Используется для отображения   преимущественно числовых значений.
<multicol> multi column Многоколоночный. Если текст содержит много колонок — задает их ширину, количество и расстояние между колонками.
<nav> navigate Проводить. Задает навигацию сайта. Допустимо использовать несколько тегов.
<nobr> no break Без разрывов. Уведомляет браузер отображать текст без переносов.
<noembed> no embed Без вставок. Предназначен для отображения веб страницы, в случае если браузер не поддерживает работу   плагинов.
<noframes> no frames Без фреймов. Содержимое данного тега отображается в браузере если он не поддерживает фрейм или не умеет их интерпретировать.
<object> object Объект. Данный элемент сообщает браузеру как загружать и отображать объекты, которые он не «понимает»
<ol> ordered list Упорядоченный список. Позволяет установить нумерованный список. Каждый элемент списка должен начинаться с тега <li>
<optgroup> option group Группа вариантов. Представляет собой контейнер, внутри которого располагаются теги <option>  объединенная в одну группу.
<option> option Вариант. Определяет отельные группы списка,  созданного при помощи тега <Select>
<output> output Вывод. Определяет область, в которую выводится информация. В основном при помощи скриптов.
<p> paragraph Параграф. Подразделение текста внутри контента. Всегда начинается с новой строки.
<param> parameter Параметр. Передает значения параметров объектам веб-страницы, созданными при помощи <applet> и <object>
<plaintext> plain text Простой текст. Отображает содержимое контейнер «как оно есть».
<pre> perfomated text Преформатированный текст.
<progress> progress Прогресс. Позволяет отображать прогресс завершенности задачи.
<q> quotation Цитата. Используется для выделения цитат в тексте.
<rp> ruby fallback text Резиновый текст Руби. Используется для вывода текста в браузерах которые не поддерживают тег <ruby>.
<rt> ruby text Текст Руби. Добавляет аннотацию сверху или снизу текста, заключенного в контейнер <ruby>.
<ruby> ruby annotation Аннотация Руби. Используется для добавления небольшой аннотации сверху или снизу от заданного текста.
<s> strike Удар. Позволяет отобразить часть текста как перечеркнутую.
<samp> sample output Выход образца. Используется для отображения текста, который является результатом вывода скрипта или компьютерной программы.
<script> script Скрипт. Предназначен для описания скриптов. Также может содержать ссылку на программу или ее текст на определенном языке.
<section> section Секция. Задает раздел документа. Также применяется для блока новостей, глав текста и прочей информации.
<select> selection Выбор. Позволяет сделать элемент интерфейса в виды раскрывающегося списка.
<small> small Маленький. Позволяет уменьшить размер шрифта на единицу по сравнению с обычным текстом.
<source> alternative media resourse Альтернативные медиа ресурсы. Вставляет аудио или видео файл для тегов <audio> и <video>
<spacer> spacer Распорка. Создает пустое пространство по вертикали или по горизонтали.
<span> span Диапазон. Предназначен для определения строчных элементов документа.
<strike> strike Аналогичен тегу <s>. Отображает текст как перечеркнутый.
<strong> strong Сильный. Предназначен для акцентирования  (выделения) части текста. В браузерах отображается жирным начертанием.
<style> syle Стиль. Предназначен для определения стилей элементов веб-страницы.
<sub> subscript Индекс. Отображает текст в виде нижнего индекса.
<summary> summary Является заголовком для тега <details>, по которому можно щелкать для сворачивания\разворачивания.
<sup> Superscript Отображает шрифт в виде верхнего индекса.
<table> Table Этот тег является контейнером для всех элементов, определяющих содержимое html таблицы.
<tbody> Table body Предназначен для хранения одной или нескольких строк таблицы.
<td> Table data Позволяет создавать одну ячейку таблицы.
<textarea> Text area Область текста. Представляет собой элемент формы для создания области, в которую можно ввести несколько строк текста.
<tfoot> Table footer Позволяет хранить одну или несколько строк внизу таблицы.
<th> Table header Создает одну ячейку таблицы которая является заголовком.
<thead> Table head Создает одну или несколько строк, которые находятся во главе таблицы.
<time> Time Текст внутри этого тега обозначается как дата или время.
<title> Title Тег, который определяет заголовок документа.
<tr> Table row Контейнер для создания строки таблицы.
<track> Track Позволяет авторам указать текстовую  дорожку для медийных файлов
<tt> Teletype Тег физического форматирования. Отображает текст моноширинным.
<u> Unarticulated Позволяет добавить подчеркивание к   тексту.
<ul> Unordered list Устанавливает маркированный список.
<var> Variable Применяется для выделения переменных компьютерных программ
<video> Video Позволяет добавлять, воспроизводить и управлять настройками видео в пределах веб-страницы.
<wbr> Word break Показывает браузеру место, где допускается сделать перенос строки текста
